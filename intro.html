Welcome to the web application for running the Itegrated Transport and Health Impacts Model (ITHIM)

This application runs the ITHIM package in R version 0.5.7 For more documentation on the package and a list of publications using ITHIM, visit the ITHIM page on GitHub.

There is also a tutorial available if you wish to run the package with increased fuctionality directly in R.


This application estimates the physical activity benefits of increased active transportation behaviors among a population. It requires three inputs, each stratified by age and sex:
  • the average time spent doing active travel (walking and cycling) each week
  • the current burden assoicated with specific chronic disease outcomes affected by physical activity
  • the number of people in each age-sex cohort
The application is preloaded with data for the Portland Metro Area, but you may also upload tables of the three variables for your area / population of interest.
